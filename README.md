# IL-2 Great Battles Modular Mission Orchestrator

A Python toolchain for generating IL-2 Great Battles multiplayer missions from modular, reusable components. Three-layer architecture: shared parser library → backend tool suite (Compiler, Reverse Engineer, Map Extractor) → user-facing GUI and CLI. Each phase produces a standalone deliverable.
